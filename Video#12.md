# 12. Gitignore
~~~
Gitignore adalah sebuah sebuah file yang bisa disimpan
di dalam repository agar kedepannya ketika kita add dan commit akan ada
file yang tidak akan terbawa add dan commit-nya.
gitignore digunakan dengan cara mengkolaborasikan Git dan Github
~~~
~~~
Kita harus mendaftarkan file yang tidak ingin
ikut di-add atau di-commit dengan gitignore. gitignore ada secara bawaan di Github.
Gitignore sangat berguna saat bekerja dengan banyak file.
~~~
