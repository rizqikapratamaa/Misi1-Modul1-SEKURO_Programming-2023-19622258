# 7. Git: Merge Conflict
~~~
Merge conflict terjadi ketika dua branch
mengerjakan baris yang sama dalam satu repository.
Maka harus diselesaikan secara manual.
Penyelesaian conflict dilakukan di
code editor yang tersedia di komputer.
~~~
~~~

~~~
