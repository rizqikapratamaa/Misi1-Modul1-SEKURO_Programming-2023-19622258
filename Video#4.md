# 4. GitHub: Fork
~~~
Forking singkatnya adalah menduplikat repository orang lain yang dapat kita
modifikasi sesuka hati tanpa
memengaruhi repository aslinya.

Forking sangat berguna untuk bekontribusi pada repository orang lain

~~~
~~~
Cara melakukan forking adalah dengan klik tombol fork secara langsung 
ataupun klik edit hingga repository terduplikat.
~~~
~~~
Pemilik asli pun bisa melakukan pull terhadap
hasil fork yang dilakukan orang lain
~~~
